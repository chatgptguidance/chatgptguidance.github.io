Cursor：集成 ChatGPT 的代码编辑器 智能写代码神器 保姆级教程

Created2024-02-29|Updated2024-02-29|OpenAI
|Post Views:

你听说过 Cursor 吗？

Cursor 是一款 AI 代码编辑器，VSCode （Visual Studio Code）开发而来，可以自动执行与编程相关的任务。使用 Cursor，不仅可以自动生成代码，还可以在聊天中提问并自动纠正错误！它的亮点是通过 GPT-4 来辅助你编程，完成 AI 智能生成代码、修改 Bug、生成测试等操作。

你可以用 Cursor 做的 3 件事

一、自动代码生成和编辑

Cursor 集成了 ChatGPT，可以自动生成和编辑编程代码。

因此 Cursor 可用于自动执行基本编码和重复性任务，使用户能够专注于更具创造性和挑战性的任务。

二、聊天中的问答

Cursor 具有 AI 驱动的聊天功能，可以向它提出任何问题。

例如，问题包括：

  • 这段代码是什么意思？
  • 如何使用 Cursor 中的这个功能？

如果你好好利用 Cursor 的聊天功能，可以减少花在其他任务上的时间，让你的编程工作更有效率。

三、自动调试和纠错

Cursor 具有自动检测编程过程中发生的错误（错误），并提出更正建议的功能。

就人类而言，长时间编写代码时，错误是不可避免的。 但是，Cursor 每次都会检测错误，因此可以大大减少不必要的更正工作。

Cursor 是否免费？

Cursor 现在已经不再完全免费使用。目前为止，Cursor 的定价计划如下：

基础版 专业版 商用版
价格 免费 $20 每月 $40 每月
GPT-3.5 使用限制 100次 无限 无限
GPT-4 使用限制 50次 500次 无限

如果只是想尝试一下 Cursor 的功能，基础版完全够用。但是，如果想在工作和学习中使用 Cursor，基础版的次数限制就显得捉襟见肘了。

但是，Cursor 可以导入自己的API key，这样就能不受使用次数的限制，假如你是 ChatGPT Plus 用户，可以直接使用 GPT-4 功能，不需要二次付费。

如果在账号注册、登录、下载软件的过程中遇到网络问题，可以参考这篇文章：url

Cursor 下载及使用说明

首先需要到官网下载安装该软件，目前它能支持的有 Windows、macOS、Linux 等主流操作系统。 下载地址：https://www.cursor.so/ 

安装完成后，进行初始设置，然后单击[continue]（如果您不讲究，可以保留默认值）。

选择[help improve cursor]或[privacy mode]（如果不处理重要数据，选择[help improve cursor]就可以了）

使用前需要登陆，可以注册账号，或者直接使用 Github 等账号登录。

安装完成之后，打开 Cursor，点击右上角的图标，打开或关闭侧边栏，开始和 AI 对话。

Cursor 目前是部分免费的，每个月有100次的限额，超出需要付费，可以导入自己的API key，这样就能不受使用次数的限制，假如你是 ChatGPT Plus 用户，可以直接使用 GPT-4 功能，不需要二次付费。

导入 API key

首先需要登录你的 OpenAI 账号，网址为https://platform.openai.com/，ChatGPT 账号注册及升级 Plus 可以参考这篇文章: url

然后点击头像，有个 View API keys

点击Create new secret key生成 API key

随便输入一个名字，点击Create secret key，创建自己的API key

点击复制，再次点击Done完成。这样 API key 就复制到剪贴板了。请妥善保管好自己的 API key，避免被白嫖

然后就可以将 API key 复制到 Cursor中了，开始享受 AI 生成代码吧！
