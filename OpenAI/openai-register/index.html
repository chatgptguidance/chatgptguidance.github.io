ChatGPT账号免手机验证码注册教程（2024年2月更新）

Created2024-02-27|Updated2024-02-29|OpenAI
|Post Views:

前置提醒

访问 ChatGPT 或者 OpenAI 对网络有较高要求，有需要的可以参考这篇文章：网络加速快速指南 一键访问ChatGPT OpenAI

一、注册 Proton 邮箱

1.1 进入 Proton 官方网站

https://proton.me/

点击「Create a free account」

1.2 点击左侧的免费套餐

1.3 填写用户名、密码，并选择邮箱后缀

推荐选择 proton.me

1.4 完成验证

依次拖动验证码

设置昵称

这里可以跳过手机验证

注册完后，可以去登陆邮箱，方便接收验证码：

https://proton.me/

二、注册 OpenAI 账号

首先打开 OpenAI 的账户注册页面，输入邮箱:

https://chat.openai.com/

首先验证邮箱，去 Proton.me 收件箱里点击「Verify email address」即可。

然后提供你的名和姓及生日，并点击继续

然后选择您创建 API 还是 ChatGPT账户即可。

个人用户一般都选择 ChatGPT 账户即可：

Openai 的用户体系分成两大类：ChatGPT 用户和API开发者用户，两个体系是相互独立的，用户名密码是一致的，但是访问入口和支付都是独立的：

ChatGPT 用户的登录入口是 https://chat.openai.com
API 开发者用户的登录入口是 https://platform.openai.com

ChatGPT 用户获取 GPT-4 的使用权限，是通过购买 GPT Plus 会员，会员的收费标准是 20 美元/每月。

购买 GPT Plus 会员后，可以在 Chat 界面选择 GPT4 进行聊天。

购买教程在这里:

ChatGPT Plus 快速升级方法

API 开发者用户获取 GPT-4 API的使用权限，是通过升级付费帐户，升级付费帐户的教程在这里。

然后需要预充值5美元以上，才能开通GPT-4的访问权限
